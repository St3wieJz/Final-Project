# Final-Project
# SIR-CIM Rumor Spreading Model Reproduction

This repository contains the code for our Final Project. We reproduce the SIR-IM model from Qiu et al. (2020) and extend it with a Truth-Spreader (T) compartment (SIR-CIM).

## Project Structure

- `scripts/`: Scripts to replicate specific figures from the paper.
- `figures/`: Output plots generated by the scripts.

## How to Reproduce
1. Open MATLAB (R2024b recommended).
2. Navigate to the `scripts/` folder.
3. Run `sir-im.m` to generate the baseline comparison.
4. Run `sir-cim.m` to see the Truth-Spreader intervention results.
